Глоссарий

Бордюрная лента – (лента бордюрная) - пористая односторонне липкая лента для ограничения области заливки фотополимера и задания высоты фотополимерной печати путем обклеивания негатива по периметру.

Демпфер – пористый двусторонне липкий материал для приклеивания печати к оснастке.

Затемнитель негатива – спрей для затемнения негатива и увеличения его оптической плотности и контрастности.

Матовая пленка – (Кимото) - матовая пленка для распечатывания негативов на лазерном принтере, за счет шероховатой поверхности тонер принтера ложится на нее более равномерно чем на прозрачную пленку.

Оранж-фильм – пленка для создания негативов контактным способом. Позволяет получать качественные негативы.

Пленка защитная – пленка, предназначенная для многократного использования негатива. Помещается между негативом и фотополимером.

Скотч двухсторонний - используется для крепления печатей к оснастке в тех случаях, когда не используется демпфер.

Твердый фотополимер - используется для изготовления клише для тиснения с помощью термического процесса.

Ультрафиолетовые лампы – лампы определенного спектра свечения для засветки жидкого и твердого фотополимера.

Фотополимер – светочувствительный материал, предназначенный для засветки ультрафиолетом. Существуют жидкие фотополимеры ( для изготовления печатей), затвердевающие послойно и твердые листовые, как правило, водовымывные фотополимеры для изготовления клише для горячего тиснения.

Наша компания предлагает Вам расходные материалы для изготовления печатей и штампов по фотополимерной технологии (жидкие водовымывные фотополимеры, твердый водовымывной фотополимер на металической основе, самоклеющуюся бордюрную ленту для ограничения области заливки фотополимера и пр.).
